# Аттестация модуль 4
не судите строго пожалуйста
Название: Трекер прочитанных книг
Описание: Консольное приложение для учёта прочитанных книг с модульной архитектурой
Структура модулей:
- main.py — точка входа и пользовательский интерфейс
- storage.py — чтение, запись и управление данными в JSON
- stats.py — расчёт средней оценки и статистики по авторам
- models.py — класс Book
Как запустить: python main.py
Пример работы: меню содержит 6 вариантов действий:
1) Добавить книгу (указать автора, название, оценку(1-5), дату прочтения)
2) Показать все книги (выводит все книги + информация)
3) Показать среднюю оценку (показывает среднюю оценку всех книг)
4) Статистика по авторам (выводит имена всех авторов, количество книг каждого и их средний рейтинг)
5) Удалить книгу (удаляет книгу по названию)
6) Выход (прерывает исполнени программы и говорит до свидания)
